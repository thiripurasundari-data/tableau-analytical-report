# Tableau Analytical Report

## Project Overview
This project analyzes sales data to identify revenue trends, top-performing products, and regional performance using Tableau.

## Tools Used
- Tableau
- SQL
- Excel

## Key Insights
- Identified highest revenue generating region
- Analyzed monthly sales growth trends
- Compared product category performance
- Evaluated profit margins across segments

- Data visualization
- KPI analysis
- Business insight generation
